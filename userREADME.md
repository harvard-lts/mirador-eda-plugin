# Using Mirador with the EDA Plugin

## Getting Started

Click the hamburger menu `Toggle sidebar` button on the upper left of the viewer to see the sidebar options. If the item in the viewer is an Emily Dickinson manuscript with transcriptions, the sidebar will have an additional "T" icon.

To close the sidebar, click the `Collapse sidebar` icon (left-facing arrow) on the tab of the top right side of the sidebar.

## Finding Metadata

1. Click the `Information` icon (i) in the window sidebar
2. The metadata panel will open, showing:
   - Title of the manuscript
   - Credits
   - Publication history
   - Library ID
   - Download button
   - Resource
   - Related links

## Viewing All Items

1. Click the `Index` icon in the window sidebar (looks like a list) to see a list of all items
2. Or, click the arrow icons at the bottom of the viewer to navigate through the items

## Transcriptions

1. If there are transcriptions for the Emily Dickinson manuscript currently in the viewer, you will see a "T" icon in the window sidebar
   - If no transcriptions exist, the icon will not be shown
   - If the item in the viewer is not an Emily Dickinson manuscripts, the icon will not be shown
2. Select an Edition:
   - Use the dropdown at the top of the panel
   - Choose between available editions (e.g., Franklin Variorum 1998, Johnson Poems 1955)
   - Each edition may have different editorial choices and annotations
3. Toggle Display Options:
   - Line Breaks: Switch between metric and physical line breaks
   - Editorial Marks: Show/hide editorial marks and textual variants

## Annotations

If there are annotations are available for a manuscript, the raw text is shown in the `Annotations` button (looks like a comment bubble) on the sidebar.

## Viewing Multiple Items

1. Click the `Add resource` button (+) in the upper left
2. Select and item from the list or add a new resource from a URL
   - Selecting an item will open in an additional window in the viewer
   - Windows can be closed with the `Close window` button on the upper right of the window

## Viewing the Manuscript
1. **Zoom**: Enable the zoom controls on the left sidebar `Workplace settings` button (gear)
   - Select `Show zoom controls` to see a `Zoom in` icon (+), a `Zoom out`, icon (-), and a `Reset zoom` icon (diagonal arrows) at the bottom of the viewer
2. **Pan**: Click and drag to move the image
3. **Rotate**: Use `r` to rotate the image clockwise and `shift + r` to rotate counterclockwise

## Troubleshooting

If you don't see the transcription ("T") icon:
1. Verify you're viewing an Emily Dickinson manuscript
2. Check if the manifest contains transcription data
3. Try refreshing the page
